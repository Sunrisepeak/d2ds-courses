# 动手写数据结构(d2ds)开源课程

## 简介

一个**任何人都可以免费(通过互联网)获取**, 并且强调**动手实践**的数据结构开源课程

- 强调**动手写代码**
- 强调**课程共建共享**

> Note: 欢迎大家积极的参与到课程的学习、讨论、反馈、和建设中, 来促进**d2ds系列**的不断优化和改善**-ing**, 为后续的参与者提供更优质的内容和学习体验
